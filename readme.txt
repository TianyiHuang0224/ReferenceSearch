Before searching：


1.Installing libs：

python pip install -r requirement.txt

2.Putting the Latex file(.tex) containing references in the directory where run.py is located, then:

python run.py

3.Typing 'a' to search in Arxiv and typing 'r' to search in Researchgate(a/r)

4.The results are saved in result_arxiv.txt and result_researchgate.txt.
